# Design

Audience: AI agents implementing brand and UI for Genuine Gossip.

These documents are derived from Claude Design exports. They are the
canonical specification for visual identity, design system, voice, and
marketing surfaces. Implement against them; do not invent.

## Files

- `brand.md` — name story, wordmark, monogram, app icon, brand palette,
  applications (OG, letterhead, avatars).
- `system.md` — design tokens (color, type, space), the six core
  components, motion, card-screen anatomy.
- `voice.md` — consolidated writing rules for in-app copy and marketing.
- `surfaces/holding-page.md` — pre-launch landing for `genuinegossip.app`.

## Status

v0.1 — May 2026. Source: Claude Design artifacts (brand book +
typographic design system + holding page).

## Conventions

- One canonical home per rule. Design tokens are defined once in
  `system.md` and referenced elsewhere.
- Names follow the brand: lowercase `genuine gossip.` with a period;
  monogram is italic `gg`. The legacy "Two of Us" name from earlier
  iterations is obsolete — ignore it if it appears in any specimen.
- The product palette is closed (bone + ink + greys). The brand
  palette (claret + champagne) is for marketing surfaces only and
  appears on at most one surface at a time.
